# Dental Harmony

Landing page institucional para uma clínica odontológica, responsiva e desenvolvida em HTML puro, Tailwind CSS e JavaScript.

## 🛠️ Tecnologias

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (para os efeitos de rolagem e carrossel)
- Lucide Icons

## 🚀 Como rodar o projeto

O projeto não possui dependências de pacotes (npm/node). Para visualizar a página:

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente em seu navegador ou utilize a extensão **Live Server** no VS Code.
